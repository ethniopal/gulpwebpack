# Gulp 4 avec Webpack
Ce repo est une base optimisé des avantages de gulp et webpack réuni.

## Comment l'utiliser
Télécharge les packages npm et execute le gulp
```
npm install
npm run dev
```


Pour réinitialiser la structure initial
```
npm run init
```

Pour la mise en production.
```
npm run build
```
